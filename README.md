# my-store-node

## install

```sh
git clone https://github.com/LucianoJavierCostaPeralta/my-store-node.git
```

```
cd my-store
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
